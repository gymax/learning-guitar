# 搞懂所有现代和弦记号

## 和弦

- 和弦 Chord: 三个音或以上的、三度堆叠的组合
- 三和弦 Triad：三个音的和弦
- 七和弦 Seventh Chord: 四个音的和弦
- 九和弦 Ninth Chord：五个音的和弦
- 十一和弦 Eleventh Chord：六个音的和弦
- 十三和弦 Thirteenth Chord：七个音的和弦
